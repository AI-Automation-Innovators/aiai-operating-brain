# Start Here

Use this file to orient quickly before working in the AI Automation Innovators operating brain.

## What this repo is

A shared knowledge base for Kevin Tunis and AI Automation Innovators. It helps AI agents and human collaborators understand the business, reuse accurate context, and avoid contradictory or unsupported claims.

## What this repo is not

- Not a web app.
- Not a software package.
- Not a marketing automation system.
- Not a place to store invented proof, fake client results, or speculative claims.

## Recommended reading order

1. `README.md` — repository map and operating principles.
2. `AGENTS.md` — rules for AI agents working in this repo.
3. `business-context/README.md` — stable business positioning.
4. `offers/README.md` — current offers and how to describe them.
5. `projects/README.md` — active work instructions and status.
6. `proof/README.md` — examples and evidence rules.
7. `ai-visibility/README.md` — public-facing AI visibility guidance.

## Regular maintenance

Update these when reality changes:

- Business positioning or geography in `business-context/`.
- Offer scope, pricing notes, or delivery language in `offers/`.
- Active project status in `projects/`.
- Verified examples and supported outcomes in `proof/`.
- Public-facing descriptions in `ai-visibility/`.

## Safe working rule

If a change would add complexity, new folders, automation, broad rewrites, or unsupported claims, pause and make the smaller clearer change instead.
